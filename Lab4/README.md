# Лабораторна робота №4  
## Візуалізація гармоніки з шумом та фільтрацією

---

## 📌 Опис роботи

Програма дозволяє будувати графік гармонічного сигналу з накладеним шумом та виконувати його фільтрацію в реальному часі.

Користувач може змінювати параметри сигналу та шуму за допомогою інтерактивного інтерфейсу.

---

## ⚙️ Функціональні можливості

- Побудова гармоніки:  
  **y(t) = A · sin(ωt + φ)**

- Накладання шуму (гауссівського)

- Інтерактивне керування:
  - амплітудою
  - частотою
  - фазою
  - параметрами шуму

- Вмикання / вимикання шуму (checkbox)

- Фільтрація сигналу (Butterworth filter)

- Відображення:
  - початкового сигналу
  - відфільтрованого сигналу

- Кнопка **Reset** для скидання параметрів

---

## 🏗 Архітектура

Основні компоненти:

- `harmonic_with_noise()` — генерація сигналу
- `apply_filter()` — фільтрація сигналу
- Інтерактивний UI:
  - Slider
  - CheckBox
  - Button
