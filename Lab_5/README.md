# Лабораторна робота №5
## Наука про дані: обмін результатами та початковий аналіз

Цей проєкт є інтерактивним веб-додатком для візуалізації та аналізу екологічних індексів (VCI, TCI, VHI) для областей України.

### Основні можливості:
* **Фільтрація:** вибір індексу, області, діапазону тижнів та років.
* **Інтерактивність:** кнопка скидання всіх фільтрів до початкового стану.
* **Візуалізація:**
  - Таблиця з відфільтрованими даними.
  - Графік часового ряду для обраної області.
  - Порівняльна діаграма середніх значень індексу по всіх областях.
* **Сортування:** можливість сортування таблиці за зростанням або спаданням значень.

### Як запустити локально:
1. Встановіть необхідні бібліотеки:
   ```bash
   pip install -r requirements.txt
   ```   

2. Запустіть додаток:
   ```bash
   streamlit run app.py
   ```
