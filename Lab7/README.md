# Лабораторна робота №7  
## Computer Vision та багатопотоковість у C++

---

## Мета роботи

Навчитися поєднувати класичне програмування на C++ із сучасними методами Computer Vision та Machine Learning.

Освоїти:
- використання модуля `cv::dnn` для детекції об'єктів
- роботу з нейронними мережами в OpenCV
- багатопоточність (`std::thread`, `std::mutex`, `std::atomic`)
- уникнення блокування інтерфейсу при важких обчисленнях

---

## ⚙️ Опис роботи

Програма зчитує відео з камери та відображає його у реальному часі.

Було додано новий режим:
- **FACE MODE** — детекція облич за допомогою нейронної мережі (ResNet-10)

Детекція виконується в окремому потоці, що дозволяє:
- зберегти плавність відео (високий FPS)
- уникнути зависання інтерфейсу

---

## 🏗 Архітектура

Проєкт має модульну структуру:

- **CameraProvider** — отримання кадрів з камери  
- **FrameProcessor** — обробка зображення  
- **KeyProcessor** — обробка клавіш та режими  
- **Display** — відображення кадру  
- **FaceDetector** — багатопоточна детекція облич  

---

## 🔄 Багатопоточність

### Main Thread (UI):
- отримує кадр з камери
- передає кадр у FaceDetector
- отримує координати облич
- малює рамки
- відображає результат

### Worker Thread:
- отримує кадр
- виконує `net.forward()`
- знаходить обличчя
- оновлює координати (захищено `mutex`)
